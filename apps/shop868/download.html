Install Shop868

Shop868 is a web app. You can add it to your home screen so it feels like a normal app.

← Back to Shop868

Use Shop868 in your browser

From the Shop868 page, tap “Open Web App” to launch the web version.

Add to Home Screen – Android (Chrome)

  1. Open Shop868 in Chrome.
  2. Tap the ⋮ menu (top-right).
  3. Tap Add to Home screen or Install app.
  4. Confirm the name (Shop868) and tap Add.
  5. Find the Shop868 icon on your home screen and open it like a normal app.

Add to Home Screen – iPhone / iPad (Safari)

  1. Open Shop868 in the Safari browser.
  2. Tap the Share icon (square with an up arrow).
  3. Scroll down and tap Add to Home Screen.
  4. Confirm the name (Shop868) and tap Add.
  5. Use the new Shop868 icon on your home screen to open the app.

Desktop / Laptop

On a computer, just open Shop868 in your browser and bookmark the page. On Chrome or Edge you can also use Install app from the browser menu to pin it to your taskbar or start menu.
